# Dotfiles

My personal configs.

## Prerequisite

The dotfiles require the following tools

- [homebrew](https://brew.sh/)
- tmux (via brew)
- vim (via brew)
- hub (via brew)
- java jdk

## Installation

Clone the repository into

    git clone git@github.com:bigcrunsh/dotfiles ~/dotfiles

Symlink dotfiles into your home directory

    cd ~/dotfiles
    ./pin
